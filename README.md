# VB-DigitalMarketing

Starterproject voor de website van VB-DigitalMarketing.

## Setup

1. Ga naar de projectmap:
   ```bash
   cd "C:\Users\danie\OneDrive\Bureaublad\vstudio site vb\VB-DigitalMarketing"
   ```
2. Installeer de dependencies:
   ```bash
   npm install
   ```
3. Start de ontwikkelserver:
   ```bash
   npm run dev
   ```

## Structuur

- `src/pages` - routes en pagina's
- `src/layouts` - basislayout en herbruikbare pagina-wrapper
- `src/components` - kleine herbruikbare UI-blokken
- `src/content/blog` - blogposts in Markdown
- `src/data/locaties.json` - lokale pagina's / locaties
- `src/styles/global.css` - Tailwind + basisstijl
- `public` - statische assets
